# CONVOTIS Target Segment Library

## Purpose

This document serves as the central audience intelligence library for the CONVOTIS AI Campaign Engine.

The purpose of this library is to provide reusable audience building blocks that can be combined to create campaign-specific Ideal Customer Profiles (ICPs).

The library should be used by:

The library should be used by:

- Briefing Assistant
- Campaign Strategist
- Content Planning & Creation Agent
- Webinar Agent
- Outreach Agent
- Future Automation Workflows

This document does not define static target audiences.

It defines reusable audience components that can be combined depending on:

- Business Unit
- Service Offering
- Business Challenge
- Campaign Objective
- Buyer Journey Stage

---

# Core Principle

Target segments should not be defined by industry alone.

Target segments should be constructed using:

text Company Size + Region + Role + Business Challenge + Buying Trigger + Business KPI + Objection + Proof Point + Content Angle
